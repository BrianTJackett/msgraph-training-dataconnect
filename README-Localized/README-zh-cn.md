# Microsoft Graph 培训模块 - Microsoft Graph 数据连接

此模块将介绍 Microsoft Graph 数据连接。

## 实验 - 使用 Microsoft Graph 数据链接进行分析以查找行业专家

在此实验中，你将使用 Microsoft Graph 数据连接来分析 Office 365 中的组织电子邮件，以查找有关特定主题的行业专家。

- [实验手册](./Lab.md)
